== Description ==

This plugin lets you add privacy policy, terms and conditions, credits, refund policy, where you want with widget, php code or a simple [shortcode].

== Installation ==

1. Upload the `ELAN42-disclaimer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it!
